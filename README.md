# Supermarket Management System

A web-based management system for small and medium-sized supermarkets, built with Python Flask and SQLite. The system provides role-based access for different staff, streamlining daily operations like sales, inventory, purchasing, and membership management.

🚀 Features

Multi‑role access – Super admin, cashier, supplier manager, product manager.

Sales & checkout – Scan items, apply member discounts, print receipts.

Inventory control – Automatic stock alerts, purchase planning, and manual adjustments.

Product management – Add, modify, delete products; set promotions and sales flags.

Membership system – Register/renew members, track spending, apply 5% discount.

Supplier management – Maintain supplier details and purchase records.

Sales analytics – View daily/monthly/yearly sales charts and rankings (powered by ECharts).

Secure access – Login required; each role sees only permitted data.

🛠️ Tech Stack

Backend: Python, Flask

Database: SQLite

Frontend: HTML, CSS, JavaScript, ECharts
